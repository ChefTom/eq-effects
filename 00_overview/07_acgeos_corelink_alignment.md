# ACGEOS and CoreLink Alignment

## Overview

EQ Effects is designed to operate as a mathematical work environment within a larger governed compute architecture.

In an ACGEOS-native deployment, EQ Effects provides the user-facing mathematical environment, while ACGEOS provides orchestration, governance, identity, audit, and lifecycle control.

CoreLink provides the execution connector between governed work objects and available compute resources.

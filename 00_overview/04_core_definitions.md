# Core Definitions

## EQ Effects
A governed mathematical execution environment where user intent is transformed into structured mathematical objects and executed by the computer with full audit and transparency.

## Multipath Conversation Builder
A flexible intake system allowing users to express mathematical intent through conceptual language, structured formulas, and contextual inputs simultaneously.

## Mathematical Object
A canonical representation of mathematical intent, including equations, variables, constraints, assumptions, and precision requirements.

## Contextual Box
A live semantic state container tracking variables, assumptions, units, constraints, and execution readiness.

## Rich Result Package
A structured output that includes final results, process trace, assumptions, validation, and replay capability.

## Computer as Solver
A system model where execution responsibility is shifted from user-defined software to governed compute orchestration.

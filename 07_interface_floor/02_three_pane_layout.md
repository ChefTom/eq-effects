# Three-Pane Interface Model

## Overview

EQ Effects uses a three-pane interface to align human reasoning with mathematical structure and execution context.

---

## Left Pane: Conceptual Conversation

- Natural language interaction
- Intent expression and refinement
- Clarifications and domain reasoning

---

## Center Pane: Structured Math

- Equation construction
- Variable definitions
- Constraints and conditions
- Symbolic and numeric forms

---

## Right Pane: Contextual Box

- Live state of variables and assumptions
- Units and dimensional checks
- Precision directives
- Execution readiness indicators

---

## Interaction Model

- Changes in one pane propagate to others
- Users can move fluidly between conceptual and formal views
- The system maintains consistency across all representations

---

## Outcome

The interface supports:
- conceptual thinking without forced coding
- structured mathematical rigor
- transparent preparation for execution

It aligns human reasoning with machine computation without requiring translation into traditional programming workflows.
